# ❤️ Surpresa de Dia dos Namorados

Um site desenvolvido como uma homenagem especial de Dia dos Namorados, criado para celebrar nossa história e registrar momentos importantes da nossa jornada juntos.

## ✨ Sobre o Projeto

Este projeto foi criado com muito carinho para apresentar uma surpresa romântica através de uma página web personalizada.

O site conta com:

- ⏳ Um contador que mostra há quanto tempo estamos juntos.
- 📸 Uma galeria com diversas fotos dos nossos momentos especiais.
- 🎵 Música de fundo para tornar a experiência ainda mais emocionante.
- ❤️ Design romântico e personalizado.
- 📱 Interface responsiva para funcionar em computadores e dispositivos móveis.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
